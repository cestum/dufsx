üöÄ How to Use the Enhanced File Explorer

  1. Start DUFS with: dufs your-directory --allow-all
  2. Open your browser and navigate to the DUFS URL (usually http://localhost:5000)
  3. Use the file tree on the left to browse your directories
  4. Click any file to instantly view or edit its content in the right panel
  5. For images: Use zoom controls, pan by dragging, or fit to screen
  6. For PDFs: View directly in browser with download option
  7. For videos/audio: Use built-in HTML5 player controls
  8. For text files: Edit with syntax highlighting and auto-save
  9. Create new files and folders using the + icons in the tree header
  10. Save changes with Ctrl+S (Cmd+S on Mac) or automatic saving
